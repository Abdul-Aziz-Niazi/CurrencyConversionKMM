# CurrencyConversionKMM

This is a Kotlin Multiplatform Mobile (KMM) project that uses the [Open Exchange Rates API](https://openexchangerates.org/) to convert currencies.

## Technologies Used
- Kotlin Multiplatform Mobile (KMM)
- Kotlin Coroutines
- Ktor
- Jetpack Compose
- SwiftUI
